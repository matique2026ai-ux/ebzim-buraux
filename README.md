# Ebzim Institutional Platform 🏛️

The official digital ecosystem for the **Ebzim Association for Culture and Citizenship** (Sétif, Algeria). This platform serves as a bridge between Algeria's rich cultural heritage and modern digital citizenship.

## 🌟 Core Features

- **Civic Reporting**: Empowering citizens to report heritage violations and contribute to the protection of national monuments.
- **Financial Sustainability**: Integrated trilingual module for project-based donations and official membership subscriptions.
- **Institutional Library**: A curated repository of archaeology research, institutional reports, and local history.
- **Admin Mission Control**: A 4-tab dashboard for executive oversight of memberships, finances, and civil reports.
- **Legal Compliance**: Fully aligned with the Association's December 2024 revised statutes.

## 🛠️ Tech Stack

- **Frontend**: Flutter (3.x) — Deep Midnight Emerald Identity system with Dark Glass aesthetics.
- **Backend**: NestJS (Node.js) — MongoDB Atlas with RBAC security.
- **Localization**: Full trilingual support (Arabic, English, French).

## 🚀 Getting Started

### 1. Prerequisites
- Flutter SDK
- Node.js (v18+)
- MongoDB Atlas account (or use development fallback)

### 2. Backend Setup
```bash
cd ebzim-backend
npm install
# Copy .env.example to .env and fill in credentials
npm run start:dev
```

### 3. Frontend Setup
```bash
flutter pub get
flutter run
```
