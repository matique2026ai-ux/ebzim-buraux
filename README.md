## 🌐 Overview | نبذة عن المشروع
EBZIM is an institutional digital platform for the **Ebzim Association for Culture and Citizenship** (Sétif, Algeria). As a member of the **UNESCO Network in Algeria**, the association focuses on cultural heritage preservation, citizenship, and national memory. This platform serves as a digital hub for members, project management (e.g., the Hamma Military Barracks restoration), and civic engagement.

منصة **إبزيم** هي الفضاء الرقمي الرسمي لجمعية إبزيم للثقافة والمواطنة (سطيف، الجزائر). بصفتها عضواً في **شبكة اليونسكو بالجزائر**، تعمل الجمعية على صون التراث الثقافي، تعزيز المواطنة، وحماية الذاكرة الوطنية. تعمل هذه المنصة كمركز رقمي للأعضاء، وإدارة المشاريع الكبرى (مثل ترميم الثكنة العسكرية بالحامة)، وتفعيل المشاركة المدنية.

---

## ✨ Key Features | المميزات الرئيسية
- 🏛️ **Heritage Projects**: Interactive timeline and details for major restoration projects.
- 📋 **Civic Reporting**: A dedicated space for citizens to report violations or theft of cultural heritage.
- 🆔 **Membership Management**: Professional digital membership workflow for association supporters.
- 📜 **Statutes & Governance**: Access to the official statutes and leadership structure.
- 🎨 **Premium UI/UX**: High-fidelity Glassmorphic design with support for Arabic, English, and French.

---

## 🛠️ Technical Stack | التقنيات المستخدمة
- **Frontend**: Flutter (Web, Android, iOS)
- **Backend**: NestJS (Node.js framework)
- **Database**: MongoDB (Atlas)
- **Authentication**: JWT & Role-based access control

## 🚀 Getting Started | البدء والتشغيل

### Prerequisites
- Flutter SDK (Latest Stable)
- Node.js & npm

### Backend Setup
```bash
cd ebzim-backend
npm install
npm run start:dev
```

### Frontend Setup
```bash
# Get dependencies
flutter pub get

# Run on Web
flutter run -d chrome
```
