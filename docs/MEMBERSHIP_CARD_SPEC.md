# EBZIM Membership Card Production Specification (Front Side)

## 1. Overview
This specification defines the visual and informational structure of the official EBZIM Digital Membership Card (Front Side). It transitions from the earlier dark glass aesthetic to a lighter, more institutional, and premium "Emerald & Ivory" design.

## 2. Visual Identity
- **Primary Palette**: 
  - **Ivory/Off-White**: Background of the card surface (#FDFCF8 or similar).
  - **Emerald Green**: Border and primary branding accents (#066B4E).
  - **Gold**: Decorative lines, logo accents, and typography highlights (#D4AF37).
  - **Slate Black**: Primary text for high legibility (#1A1A1A).
- **Typography**: 
  - **Heritage/Display**: 'Aref Ruqaa' (Arabic) for association name and institutional headers.
  - **Functional**: 'Outfit' or 'Inter' for data fields and identifiers to ensure maximum clarity.
- **Corner Radius**: 24px - 32px (Premium rounded feel).

## 3. Layout Zones (Front)
- **Header Zone**:
  - Top edge: Bilingual Association Name ("EBZIM - جمعية إبزيم للثقافة والمواطنة").
  - Center: Large official EBZIM logo.
- **Title Zone**:
  - Below logo: "MEMBERSHIP CARD / بطاقة العضوية".
- **Member Data Zone (Main)**:
  - Left: Formal photo placeholder (or real user photo) with gold border.
  - Right: Stacked data fields with bilingual labels (Labels in green/slate, data in bold black).
    - FULL NAME / الاسم الثلاثي للعضو
    - MEMBER ID / الرقم التعريفي للبطاقة
    - ISSUE DATE / تاريخ الإصدار & EXPIRY DATE / تاريخ الانتهاء (Side-by-side or stacked).
- **Security/Action Zone**:
  - Bottom Right: QR Code area for verification scans.
  - "SCAN ME / مسح ضوئي" label.

## 4. Hierarchy & Balance
1. **Logo & Identity**: The EBZIM logo remains the core anchor.
2. **Member Name**: Must be the most prominent piece of personal data.
3. **QR Code**: Positioned for easy access without disrupting the aesthetic.

## 5. Access Control
- **Visibility**: Only users with an approved membership status (`MEMBER`, `ADMIN`, `AUTHORITY`) can view their official card.
- **Non-Members**: Registered users with `PUBLIC` status will see a membership application CTA instead of this card.

## 6. Globalization
- All labels use structured bilingual (AR/EN or AR/FR) text based on the user's primary locale choice, but the card design stays consistent to ensure a unified "Institutional" look.
- Numerals: Western digits only (0-9).
