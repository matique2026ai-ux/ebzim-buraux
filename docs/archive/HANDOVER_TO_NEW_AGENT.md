# ملف التسليم للوكيل الجديد (HANDOVER TO NEW AGENT)

مرحباً بك أيها الوكيل الجديد. هذا الملف صُمم خصيصاً لكي تفهم المشروع بأكمله من الصفر في 60 ثانية لتبدأ العمل مباشرة دون تكرار أي أخطاء أو مهام سابقة.

---

## 1. هوية المشروع (Project Identity)
- **الاسم:** تطبيق جمعية إبزيم للثقافة والمواطنة (Ebzim Association for Culture and Citizenship).
- **الهدف:** منصة رقمية ثقافية للتعريف بالتراث، وإدارة انخراط الأعضاء، وعرض أخبار وأنشطة الجمعية بمدينة سطيف.
- **اللغات:** التطبيق يدعم العربية (بشكل أساسي RTL)، الفرنسية، والإنجليزية.

## 2. الهيكلة التقنية (Architecture & Tech Stack)
المشروع مبني بهيكلة منفصلة تماماً (Micro-services style) وبقواعد صارمة:
- **الواجهة الأمامية (Frontend):** Flutter Web. (تعمل حالياً عبر ملف `serve_static.js` لتصدير البناء النهائي Static Release على المنفذ `5000`).
- **الخادم الخلفي (Backend):** NestJS (Node.js). يعمل على المنفذ `3000` (موجود داخل مجلد `ebzim-backend/`).
- **قاعدة البيانات (Database):** سحابة **MongoDB Atlas** الحقيقية (لا تستخدم أي قاعدة بيانات محلية Mock/Memory، ولا تستخدم Docker أبداً). الاتصال السحابي يعمل بكفاءة 100%.

## 3. ما تم إنجازه مسبقاً (حالة التصميم الحالية)
يرجى العلم أن المهام التالية منتهية وتعمل بشكل ممتاز، ولكن **يمكنك التعديل عليها أو إعادة تصميمها إذا طلب منك المستخدم ذلك صراحة**:
1. **التصميم العام (UI/UX):** تم تطبيق تصميم زجاجي فاخر (Dark Glassmorphism) مع الألوان الخضراء والذهبية للجمعية.
2. **شاشات التطبيق:**
   - **الرئيسية (Dashboard):** منتهية، تم تفريق عرضها بذكاء بين المستخدم العادي والعضو الرسمي.
   - **الملف الشخصي والانخراط (Profile & Membership):** تم تصميم تدفق الانخراط، البطاقة الرقمية، والتسلسل الهرمي للملف الشخصي بنجاح.
   - **صفحة عنا (About Us):** تم بناء الصفحة باستخدام نصوص قانونية رسمية (مصطلح "المكتب التنفيذي" "Executive Office" مستعمل حالياً).
3. **مشكلة الاتصال بالسحابة (IP Block):** واجهنا حظر IP من قِبل MongoDB Atlas سابقاً، وتم حله من طرف المستخدم. النظام متصل الآن بنجاح.

## 4. كيفية تشغيل المشروع (How to Run)
إذا احتجت لإعادة تشغيل النظام لاحقاً، اتبع الترتيب التالي حصراً:
1. افتح موجه الأوامر داخل `ebzim-backend/` وشغل: `npm run start:dev` (المنفذ 3000).
2. افتح موجه الأوامر في مسار المشروع الرئيسي وشغل: `node serve_static.js` (المنفذ 5000).
